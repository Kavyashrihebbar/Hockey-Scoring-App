# Hockey Scoring App

The Hockey Scoring App is a simple and interactive application designed to help users track scores, teams, and match details in real time.
It provides an easy-to-use interface for creating teams, adding players, and creating matches.

## Features
-User Authentication & Access Control 
- Add and manage hockey teams
- Live score updates
- Match Analytics & Leaderboards
- Player statistics
- Responsive UI

## Technologies Used
- Java with XML (Android) 
-  Java 
- Firebase Firestore Database 
- Firebase Authentication

## Future Improvements
- Allow users to comment or chat in real time during an ongoing match.
- Enable voice input for recording goals or adding comments using speech recognition.
- Improve layout design, add animations, and make the interface more user-friendly.
- Generate and share match reports or score summaries as PDFs.
